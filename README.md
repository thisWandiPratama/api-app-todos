# Api App Todos
 Silakan clone repository 

```
git clone https://github.com/thisWandiPratama/api-app-todos.git
```

Kemudian jalan perintah berikut:

```
npm install
```
Kemudian lakukan import database

Dan jalankan server

```
node index.js
```
